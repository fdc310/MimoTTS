# MimoTTS

LangBot 插件，调用小米 MiMo TTS V2.5 API 将文本转换为语音。

## 功能

- **自动语音合成**：LLM 回复自动转为语音发送
- **命令语音合成**：通过 `!tts` 命令手动合成语音
- **LLM 文本加工**：可配置 LLM 模型对回复进行口语化处理，提升语音自然度
- **多模型支持**：MiMo V2.5 TTS（预置音色）/ 音色设计 / 音色复刻
- **多音色支持**：9 种官方预置音色可选
- **丰富风格控制**：支持情绪、语调、方言、角色扮演等多种风格
- **音色设计**：通过文本描述自定义音色
- **唱歌模式**：支持唱歌风格合成

## 安装

1. 将本插件放入 LangBot 的插件目录
2. 在 LangBot 管理面板中启用插件
3. 前往 [api.xiaomimimo.com](https://api.xiaomimimo.com) 获取 API Key
4. 在插件配置中填入 API Key

## 配置项

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `api_key` | string | 空 | MiMo TTS API Key |
| `enable_auto_tts` | boolean | true | 自动将 LLM 回复转为语音 |
| `block_text_reply` | boolean | true | 拦截文字回复，仅发送语音。关闭则同时发送文字和语音 |
| `llm_model` | llm-model-selector | 空 | 文本加工 LLM 模型，留空则跳过加工直接合成 |
| `default_model` | select | mimo-v2.5-tts | 默认模型 |
| `default_voice` | select | mimo_default | 默认音色 |
| `default_format` | select | wav | 音频格式（wav / mp3 / pcm16） |
| `default_style` | string | 空 | 默认风格，如：开心、东北话、唱歌 |

### 可用模型

| 模型 ID | 说明 |
|---------|------|
| `mimo-v2.5-tts` | 预置音色合成（推荐） |
| `mimo-v2.5-tts-voicedesign` | 音色设计（通过文本描述定制音色） |
| `mimo-v2.5-tts-voiceclone` | 音色复刻（基于音频样本复刻音色） |

## 命令

| 命令 | 说明 |
|------|------|
| `!tts <文本>` | 使用默认配置合成语音 |
| `!tts voice <音色> <文本>` | 指定预置音色合成 |
| `!tts clone <名称> <文本>` | 使用已保存的克隆音色合成 |
| `!tts clone add <名称> <URL>` | 保存克隆音色 |
| `!tts clone list` | 显示已保存的克隆音色 |
| `!tts clone remove <名称>` | 删除克隆音色 |
| `!tts style <风格> <文本>` | 指定风格合成 |
| `!tts design <音色描述> <文本>` | 音色设计合成 |
| `!tts director <模板> <文本>` | 导演模式合成 |
| `!tts director list` | 显示预设导演模板 |
| `!tts director custom <名> <角色> <场景> <指导> <文本>` | 自定义导演模式 |
| `!tts voices` | 显示预置音色列表 |
| `!tts styles` | 显示可用风格列表 |
| `!tts help` | 显示帮助信息 |

### 示例

```
# 基础合成
!tts 你好世界

# 指定预置音色
!tts voice 冰糖 今天天气真好
!tts voice Chloe Hello, how are you?

# 克隆音色管理
!tts clone add 妈妈 https://example.com/voice.mp3
!tts clone add 爸爸 data:audio/mpeg;base64,XXX...
!tts clone list
!tts clone remove 爸爸

# 使用克隆音色
!tts clone 妈妈 你好呀
!tts clone 妈妈 今天天气真好

# 指定风格
!tts style 开心 我太高兴了
!tts style 东北话 你这人可太有意思了
!tts style 唱歌 原谅我这一生不羁放纵爱自由

# 音色设计
!tts design 一位年轻活泼的女性，声音甜美清亮 你好呀

# 导演模式
!tts director 冰山美人 你好，好久不见
!tts director 热血少年 我绝不会放弃！
!tts director list
!tts director custom 我的角色 一个神秘的老人 在古寺里 高深莫测的语气 你好
```

### 克隆音色配置

在插件配置页面的「克隆音色」栏，每行填写一个音色，格式为 `名称=URL`：

```
妈妈=https://example.com/voice1.mp3
爸爸=data:audio/mpeg;base64,XXX...
老师=https://example.com/voice3.wav
```

也支持 JSON 格式：
```json
{"妈妈": "https://example.com/voice1.mp3", "爸爸": "https://example.com/voice2.mp3"}
```

### 导演模式

导演模式通过三个维度全方位刻画人物与声线：

#### 预设模板

| 模板名 | 说明 |
|--------|------|
| 冰山美人 | 冷淡疏离的御姐音 |
| 热血少年 | 明亮有力的少年音 |
| 温柔姐姐 | 温柔甜美的女声 |
| 腹黑反派 | 优雅却阴冷的贵族音 |
| 天真萝莉 | 稚嫩可爱的童声 |
| 沉稳大叔 | 低沉醇厚的成熟男声 |
| 傲娇大小姐 | 高傲却傲娇的少女音 |
| 神秘魔法师 | 空灵飘渺的神秘音 |

#### 自定义导演模板

在插件配置页面的「自定义导演模板」栏，每行填写一个模板，格式为 `名称=角色|场景|指导`：

```
我的角色=一个神秘的老人|在古寺里打坐|高深莫测的语气，语速缓慢
冰女杀手=冷酷的女杀手|在执行任务时|冰冷无情的声音，没有任何感情波动
```

或者使用命令：
```
!tts director custom 我的角色 一个神秘的老人 在古寺里打坐 高深莫测的语气，语速缓慢 你好
```

## 风格控制方式

### 1. 自然语言控制（推荐）

在 user 消息中传入自然语言描述，模型会理解并生成对应风格：

```
!tts style 用轻快上扬的语调，带着查到成绩后压抑不住的激动与小骄傲 你好
```

### 2. 标签控制

在文本开头添加风格标签：

```
!tts (开心)我太高兴了
!tts (东北话)哎呀妈呀，这天儿也忒冷了吧！
!tts (唱歌)原谅我这一生不羁放纵爱自由
```

### 3. 导演模式

更精细的角色-场景-指导控制：

```
!tts design 【角色】百年门阀岑家的现任大当家。【场景】在祠堂的阴影里。【指导】冰冷、慵懒却极具威压的低音御姐。 你好
```

## 工作流程

1. 用户发送消息 -> LLM 生成回复
2. 插件拦截回复文本，清理思维链内容
3. 若配置了 LLM 加工模型，将文本转为口语化表达并添加风格标签
4. 若未配置加工模型，使用内置规则引擎清理 Markdown、emoji 等
5. 注入默认风格
6. 调用 MiMo TTS V2.5 API 合成语音
7. 将语音以 Voice 消息发送
